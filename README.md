# BGP-Labs
This repository contains the labs completed while studying BGP (Border Gateway Protocol) during the CCNP Enterprise certification. The labs are designed to provide hands-on experience with BGP configuration, troubleshooting, and optimization in various network scenarios.

# Lab 1: Basic BGP Configuration
Lab 1 focuses on the basic configuration of BGP, including establishing BGP sessions between routers, configuring BGP neighbors, and advertising networks.

## Lab Topology


## Tasks
1. Configure BGP on all routers in the network.
2. Establish BGP sessions between the routers.
3. Advertise networks into BGP.
4. Verify BGP configuration and routing tables on all routers.
5. Test failover capabilities by shutting down one of the routers and observing how the other routers adjust their routing tables.
6. Document the configuration.
